# battleship_react
Battleship Implementation in React

### Do once in Terminal: 
git clone https://github.com/mattcarpowich1/battleship_react.git

### When you want to start it
cd client  
yarn  
yarn start  
open up browser at http://localhost:3000

### Press Ctrl-C to cancel the process
